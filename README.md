# 🚀 Quick Start

## 1️⃣ Clone Repository

```bash
git clone https://github.com/Nadhilken/html-to-apk.git
```

---

## 2️⃣ Paste Your HTML

Replace

```python
html_content = """
YOUR HTML HERE
"""
```

inside

```
Lines 109 → 861
```

---

## 3️⃣ Customize APK

```python
project_name = "MyApp"

package_name = "com.example.myapp"

icon_path = r"C:\Users\YourName\Downloads\icon.png"
```

---

## 4️⃣ Change App Name

```xml
<string name="app_name">My APK</string>
```

Replace

```
My APK
```

with

```
Your App Name
```

---

## 5️⃣ Run

```bash
python app.py
```
